# 本地AI写作助手

一个基于PyQt5和Ollama的本地AI写作助手软件。

## 功能特点

- 支持单篇和批量文章生成
- 模板管理功能
- 生成记录管理
- 在线更新功能
- 美观的用户界面

## 使用说明

1. 确保已安装Ollama并启动服务
2. 下载并运行 `本地AI写作助手.exe`
3. 选择本地模型
4. 输入写作指令或加载模板
5. 输入文章标题
6. 点击生成按钮开始生成文章

## 系统要求

- Windows 10/11
- 已安装Ollama服务
- 至少4GB内存
